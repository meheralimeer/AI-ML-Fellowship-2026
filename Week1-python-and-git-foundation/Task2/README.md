# Task 2 - Modular Python, Data Handling & File Operations

## Overview
Python programming focused on advanced functions, file handling, exception handling, and data structures.

## Files

### Core Modules
- `utils.py` - Utility functions (*args, **kwargs, lambda)
- `contact_manager.py` - File-based contact management (Read/Write/Append)
- `exception_handling.py` - Exception-safe calculator (try/except/else/finally)
- `student_records.py` - Student record system (Dictionaries, Lists, Comprehensions)
- `data_op.py` - Data manipulation (Remove duplicates, Sort, Statistics)

## Concepts Covered
- **Functions**: *args, **kwargs, lambda functions
- **File Handling**: Read, Write, Append operations with JSON
- **Exception Handling**: try/except/else/finally blocks, custom exceptions
- **Data Structures**: Lists, Dictionaries, Sets
- **Comprehensions**: List and Dictionary comprehensions
- **Complexity**: Basic time and space complexity awareness
